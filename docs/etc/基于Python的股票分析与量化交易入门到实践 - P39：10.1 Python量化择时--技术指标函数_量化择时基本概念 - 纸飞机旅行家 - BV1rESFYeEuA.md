它只是反映了一个有趋势概念的事情，只是未来可能是，但它是不是底或者是不是价格底部，不考虑，因为在实际情况里面，你很难每一次都踩中顶，就是买在价格的底，然后卖在那个价格高，这是基本上是不可能做到的。

所以在趋势指标这个理论里面，他就不考虑去捕捉顶部和特点，然后常见的趋势指标有均线指标，其实就ma macd指标，金叉模型等等，这是趋向指标。
